## Latest Release Notes

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for teams editing docs simultaneously.
- Introduced version pinning for stable documentation links.

## Improvements

- Enhanced search with fuzzy matching and code snippet highlighting.
- Optimized build times by `{30%}` for large doc sites.

## Bug Fixes

- Fixed rendering issues with nested `{<CodeGroup>}` components.
- Resolved mobile sidebar collapse on tab switch.
