## Account and Setup Issues

MOHD SHABAN makes it easy to set up your documentation space. Follow these steps if you encounter common account problems.

<Callout kind="tip">
  Always use a strong password containing letters, numbers, and symbols for your account security.
</Callout>

<Steps>
  <Step title="Create Account" icon="user-plus">
    Visit `https://app.mohdshaban.com/signup` and enter your email and password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification link and click it to activate your account.
  </Step>
  <Step title="Reset Password" icon="key">
    Go to the login page, click "Forgot Password?", and follow the emailed reset instructions.
  </Step>
</Steps>

<Expandable title="What if I don't receive the verification email?">
  Check your spam folder. If missing, request a new one from the account settings page.
</Expandable>

## Content Management Tips

Manage your docs efficiently with MDX. Use version control and previews to keep content organized.

<Tabs>
  <Tab title="Adding Pages" icon="plus">
    Create new pages via the dashboard. Use YAML frontmatter for metadata:

    ```mdx
    ---
    title: My Page
    description: Page summary
    ---
    ```

    Preview changes before publishing.
  </Tab>
  <Tab title="Version Control" icon="git-branch">
    Connect your GitHub repo for automatic syncs.

    <CodeGroup tabs="GitHub, GitLab">
    ```yaml
    # .github/workflows/docs.yml
    name: Deploy Docs
    on: push
    jobs:
      deploy:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v4
          - run: npm run build
    ```
    ```yaml
    # .gitlab-ci.yml
    deploy_docs:
      stage: deploy
      script:
        - npm ci
        - npm run build
    ```
    </CodeGroup>
  </Tab>
</Tabs>
